Introduction

The Fretboard Memorization Workshop is a JavaFX desktop application designed to help guitar players improve their knowledge of the fretboard. Through interactive drills and visual displays, you can train yourself to instantly recognize and locate any note on the guitar neck.

🎯 Interactive Drills

Practice finding notes with timed exercises that track your progress and improvement over time.

🎵 Audio Detection

Connect your guitar and the app will detect the notes you play in real-time using pitch detection.

📊 Progress Tracking

Save your practice sessions and track your improvement with detailed history and statistics.

⚙️ Customizable

Configure the app to match your guitar setup - from 6 to 8 strings, 12 to 36 frets, and more.

Getting Started

System Requirements

  • Java: JDK 25 or newer
  • Operating System: Windows, macOS, or Linux
  • Audio Input: Microphone or audio interface (for drills)
  • Screen Resolution: Minimum 800x600 pixels

Installation

  1. Build the application
    Open a terminal in the project directory and run: 
    mvn clean javafx:jlink
  2. Navigate to the output folder
    After building, go to: target/FretboardApplication/bin
  3. Run the application
    • Windows: Double-click FretboardApplication.bat
    • Linux/macOS: Run ./FretboardApplication in terminal

💡 Quick Run Option

For development or quick testing, you can run directly with Maven:

mvn javafx:run

User Profiles

User profiles store your settings, guitar configuration, and training progress. Creating a profile allows you to save your data between sessions.

Creating a New Profile

  1. Go to File → New Profile from the menu bar
  2. Enter your name when prompted
  3. Choose a location to save your profile data
  4. Configure your guitar settings (frets, strings, etc.)

Opening an Existing Profile

  1. Go to File → Open Profile
  2. Navigate to your saved profile file
  3. Select the file and click Open

Saving Your Progress

  • Save (Ctrl+S): Saves to the current profile file
  • Save As: Saves to a new location

⚠️ Important

Training modules that track progress (Random Note String Drill, Fretboard Memory Drill) require an active profile. Create or open a profile before starting these drills.

Training Modules

The application includes several training modules to help you master the fretboard. Select a module from the left sidebar to begin.

1. Fretboard Display

📍 Overview

A visual reference tool that displays your guitar's fretboard based on your settings. Use this to study note positions and familiarize yourself with the layout.

Features:

  • Show Notes: Toggle to display note names on each fret position
  • Zoom Controls: Adjust the fretboard size for better visibility
  • Wood Grain: Visual customization of fretboard appearance

How to Use:

  1. Select "Fretboard Display" from the module list
  2. Use the "Show Notes" checkbox to reveal note positions
  3. Study the patterns and note relationships across strings

2. Random Note String Drill

🎯 Overview

Practice finding all octaves of a note on individual strings. This drill works through each string from the lowest to highest, presenting random notes for you to locate.

How It Works:

  • You'll be shown 10 random notes for each string
  • Find and play ALL octaves of each note on the current string
  • The app detects your played notes via audio input
  • Goal: Complete each string in under 25 seconds

Controls:

Start Drill Stop

Tips for Success:

  • Start slow and focus on accuracy before speed
  • Learn the octave patterns - they repeat every 12 frets
  • Practice one string at a time until comfortable

3. Fretboard Memory Drill

🧠 Overview

The ultimate fretboard memorization challenge. Find ALL instances of a note across the entire fretboard, spanning all strings and frets.

How It Works:

  • You'll be shown 10 random notes
  • For each note, find EVERY occurrence on the entire fretboard
  • The display shows how many locations remain to find
  • Timer tracks your speed for each note

Controls:

Start Drill Stop

Progress Tracking:

  • Previous attempts are displayed in the history panel
  • Your progress is saved to your profile automatically
  • Review your history to track improvement over time

Audio Input Setup

For the training drills to detect your playing, you need to configure audio input properly.

Connecting Your Guitar

  1. Direct Connection
    Connect your guitar to an audio interface, then connect the interface to your computer via USB.
  2. Configure Audio Settings
    Go to Edit → Audio Settings and select your audio input device.
  3. Test the Connection
    Play some notes and verify the application responds to your input.

Audio Settings

Setting Default Description
Sample Rate 44100 Hz Audio sample rate (higher = better quality)
Buffer Size 4096 Audio buffer size (lower = less latency)
Input Device System Default Select your audio interface or microphone

💡 Using a Microphone

If you don't have an audio interface, you can use a microphone to pick up your guitar's acoustic sound. Position the microphone close to the sound hole for best results. This works best in a quiet environment.

Tips for Effective Practice

🎯 Start with the Basics

Begin with the Fretboard Display module to study note positions before attempting timed drills. Focus on learning natural notes (no sharps/flats) first.

⏱️ Practice Regularly

Short, daily practice sessions (10-15 minutes) are more effective than occasional long sessions. Consistency is key to memorization.

📈 Track Your Progress

Use the history panel to monitor your improvement. Aim to beat your previous times while maintaining accuracy.

🎸 Apply to Real Playing

As you practice, try to apply your knowledge when playing songs. Identify notes as you play them to reinforce your learning.

Suggested Practice Routine

  1. Warm-up (2 min): Review the fretboard display with notes shown
  2. String Drill (5 min): Complete one round of Random Note String Drill
  3. Full Fretboard (5 min): Attempt the Fretboard Memory Drill
  4. Cool-down (3 min): Play freely, naming notes as you play them

Mastery Goals

Level String Drill Fretboard Drill
Beginner < 60 seconds per string < 3 minutes per note
Intermediate < 30 seconds per string < 90 seconds per note
Advanced < 25 seconds per string < 60 seconds per note
Expert < 15 seconds per string < 30 seconds per note

Troubleshooting

Audio not detected

  • Verify your audio device is connected and recognized by your system
  • Check that the correct input device is selected in Audio Settings
  • Ensure no other application is using the audio input
  • Try increasing the buffer size if you experience dropouts

Notes not recognized correctly

  • Play notes cleanly and let them ring
  • Reduce background noise in your environment
  • Ensure your guitar is in tune
  • Try adjusting your playing position relative to the microphone

Application won't start

  • Verify Java 25+ is installed: java -version
  • Ensure JavaFX is properly configured
  • Try rebuilding: mvn clean javafx:jlink
  • Check console output for error messages

Training modules are disabled

  • Some modules require an active user profile
  • Create a new profile or open an existing one via File menu
  • Ensure your profile has been saved at least once
